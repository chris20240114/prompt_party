# PromptParty-Demo
Preparation for Open Project Project PromptParty. 

## Setting up your Environment
We will be using the conda package manager ([more information about conda package manager](https://anaconda.org/anaconda/conda)). Please create the environment from the environment.yml page. Please update the environment using the requirements text. 


*Setting up the Environment for the first time*
```bash
conda env create -f environment.yml 
conda activate prompt-party
```

*Updating the environment with new packages*

Note: make sure your are in the root directory.

```bash
pip install -r requirements.txt
```
